# 宝可梦文字冒险

一款基于网页的宝可梦文字 RPG 游戏，使用纯 HTML/CSS/JavaScript 开发，无需任何后端服务器，打开浏览器即可游玩。

## 游戏简介

在关都地区开始你的宝可梦训练师之旅！从真新镇出发，收集 8 枚徽章，挑战四天王，成为宝可梦联盟冠军！

### 核心玩法

- **探索地图**：穿越关都地区的城镇、道路、洞穴和水道
- **捕捉宝可梦**：在野外遭遇宝可梦，使用精灵球捕捉
- **培养队伍**：升级、学习技能、进化你的宝可梦
- **挑战道馆**：击败 8 位道馆馆主，收集徽章
- **对战训练家**：与野生训练家和劲敌小茂一决高下
- **探索七之岛**：通关后前往火红/叶绿的七之岛地区

### 游戏特色

- **完整的关都地图**：包含所有原版城镇、道路和特殊地点
- **原版技能系统**：400+ 技能，29 种效果（睡眠、麻痹、中毒、灼伤、混乱等）
- **精英野生宝可梦**：8% 概率遇到等级更高的精英野生宝可梦，击败获得 1.5 倍经验
- **道馆挑战系统**：8 位道馆馆主 + 四天王 + 冠军小茂
- **宝可梦图鉴**：记录已发现的宝可梦
- **本地存档**：游戏进度自动保存在浏览器本地存储中

## 快速开始

### 运行方式

1. 克隆仓库：
   ```bash
   git clone https://github.com/kilng235/pokemon-text-game.git
   ```

2. 直接用浏览器打开 `index.html` 即可游玩

> **提示**：推荐使用 Chrome 或 Firefox 浏览器获得最佳体验

### 操作说明

- 点击按钮进行移动、攻击、使用道具等操作
- 在探索界面可以查看队伍、背包、图鉴
- 战斗中可以选择攻击、换宠、使用道具或逃跑

## 游戏机制

### 野生宝可梦等级

各区域的野生宝可梦等级是固定的，按难度递增：

| 区域 | 等级范围 | 对应道馆 |
|---|---|---|
| 1-2 号道路 | 2-7 | — |
| 常青森林 | 4-8 | — |
| 3-4 号道路 | 8-16 | 小刚（岩石） |
| 5-6 号道路 | 14-22 | 小霞（水）、马志士（电） |
| 7-8 号道路 | 18-26 | 莉佳（草） |
| 9-10 号道路 | 20-30 | 娜姿（超能）、阿桔（毒） |
| 冠军之路 | 34-42 | — |
| 华蓝洞穴 | 50-55 | — |

### 精英野生宝可梦

- **出现概率**：8%
- **等级加成**：比该区域普通野生高 5-10 级
- **经验加成**：击败获得 1.5 倍经验值
- **视觉标记**：战斗界面显示红色"精英"标签

### 技能效果系统

游戏实装了 29 种技能效果：

| 效果类型 | 具体效果 |
|---|---|
| **异常状态** | 睡眠、麻痹、中毒、灼伤、混乱 |
| **降低对手** | 攻击↓、防御↓、特攻↓、特防↓、速度↓、命中↓ |
| **提升自身** | 攻击↑、防御↑、特攻↑、特防↑、速度↑、回避↑ |
| **组合效果** | 攻击+防御↑、攻击+速度↑、攻击+特攻↑、防御+特防↑ |
| **特殊效果** | 回复 HP、吸血、寄生种子、定身、清除能力变化 |

## 开发说明

### 数据修改

- **宝可梦数据**：编辑 `js/data.js` 中的 `POKEMON` 数组
- **技能数据**：编辑 `js/data.js` 中的 `MOVES` 数组
- **地图数据**：编辑 `js/map.js` 中的 `LOCATIONS` 对象
- **道馆数据**：编辑 `js/map.js` 中的 `GYM_LEADERS` 对象

### 添加新宝可梦

在 `js/data.js` 的 `POKEMON` 数组中添加新条目：

```javascript
[id, '名称', '属性1,属性2', HP, 攻击, 防御, 特攻, 特防, 速度, 捕获率, 基础经验, [进化等级, 进化目标ID], [[技能ID, 学习等级], ...]]
```

### 添加新技能

在 `js/data.js` 的 `MOVES` 数组中添加新条目：

```javascript
[id, '名称', '属性', 威力, PP, '描述', '效果标识符']
```

效果标识符包括：`sleep`、`paralyze`、`poison`、`burn`、`confuse`、`atkDown`、`defDown`、`recover` 等。
