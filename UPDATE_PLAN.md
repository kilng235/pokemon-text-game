## 二、动态等级缩放

### 2.1 当前问题

```javascript
// js/battle.js startWildBattle()
const [min, max] = pool.lv  // 固定范围，如 [2,6]
const level = min + Math.floor(Math.random() * (max - min + 1))
```

玩家等级低时，可能遇到等级过高的野生宝可梦。

### 2.2 改进方案

```javascript
function getScaledLevel(baseMin, baseMax) {
  const playerMax = Math.max(...G.player.pokemon.map(p => p.level), 1)
  const center = (baseMin + baseMax) / 2
  const bias = (playerMax - center) * 0.4
  const adjMin = Math.max(2, Math.round(baseMin + bias))
  const adjMax = Math.max(adjMin, Math.min(Math.round(baseMax + bias), playerMax + 3))
  return adjMin + Math.floor(Math.random() * (adjMax - adjMin + 1))
}
```

### 2.3 缩放规则

| 玩家最高等级 | 路线基础范围 | 实际遇到范围 |
|-------------|-------------|-------------|
| Lv.5 | [2,6] | [2,5] |
| Lv.10 | [2,6] | [4,8] |
| Lv.20 | [10,15] | [12,18] |
| Lv.30 | [15,20] | [18,23] |
| Lv.50 | [30,38] | [35,43] |

### 2.4 修改文件

- `js/battle.js` — `startWildBattle()` 使用缩放函数

## 三、补全红蓝版缺失剧情

### 3.1 高优先级（主线核心）

#### 圣安奴号
- 新增地点：`ssAnne`（枯叶市港口）
- 触发条件：完成海角小屋任务获得船票
- 剧情：船上训练家对战 → 船长给HM01居合术
- 战斗：5-6个水手训练家

#### 双子岛
- 新增地点：`seafoamIslands`（20号水道分支）
- 野生宝可梦：急冻鸟（Lv.50，传说宝可梦）
- 需要冲浪术才能进入

#### 无人发电站
- 新增地点：`powerPlant`（10号道路附近）
- 野生宝可梦：闪电鸟（Lv.50，传说宝可梦）
- 大量电系宝可梦

#### 红莲镇宝可梦屋
- 新增地点：`pokemonMansion`（红莲镇内）
- 剧情：坂木的实验记录、化石复活
- 道具：秘密钥匙（开道馆门）

#### 华蓝洞穴
- 新增地点：`ceruleanCave`（华蓝市左侧）
- 野生宝可梦：超梦（Lv.70，二周目）
- 需要集齐8徽章才能进入

#### 月见山化石选择
- 新增剧情事件：月见山深处选择菊石兽或化石盔
- 选择分支影响后续图鉴收集

### 3.2 中优先级（丰富体验）

- 12-18号道路补全（单车道等）
- 常青森林扩展（更多训练家）
- 岩石隧道（独立区域）
- 狩猎地带
- 道馆赛后对话
- 更多NPC对话

### 3.3 低优先级（锦上添花）

- 隐藏道具系统
- 二周目系统
- 图鉴完成奖励
